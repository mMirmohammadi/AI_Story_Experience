# LifeOS — A Simulation in Three Acts

An immersive, scroll-driven narrative experience exploring what happens when we outsource so much thinking to AI that it starts scripting us.

## View the Experience

Open `index.html` in a modern browser. Enable audio for the full experience.

## Controls

- **Scroll** — Progress through the narrative
- **Click audio button** or press **M** — Toggle sound
- **Space** — Scroll down
